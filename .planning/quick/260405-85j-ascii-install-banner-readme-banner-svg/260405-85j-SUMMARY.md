## What Was Built

### Task 1 — printBanner() in scripts/install.js

Added `printBanner()` function that runs at the end of `main()` after installation completes:

- ANSI 93 (bright yellow) box header with `(*>ω<) code-cheer v{VERSION}`
- ANSI 32 (green) status fields: character name, install path, statusline active
- Random tagline from `INSTALL_DIR/vocab/{character}[.en].json` `triggers.random`
- Language-aware: `config.language === 'en'` reads `.en.json` vocab and prints English tip line
- Full graceful fallback: vocab read failure silently skipped; outer try/catch ensures banner failure never exits non-zero
- Replaced hardcoded `console.log('(*>ω<) Nova: 安装完成！...')` block in main()

### Task 2 — scripts/gen-banner.js + assets/banner.svg

Created zero-dependency Node.js script that generates a 1280×240 SVG README hero banner:

- Dark background `#111111`
- Left: Nova kaomoji `(*>ω<)` in Nova yellow `#f4a900` at 48px
- Centre: project name `code-cheer` bold 28px + subtitle + feature line
- Right: character selector (nova highlighted, luna/mochi/iris dimmed)
- All styles inline (fill, font-size, font-weight) — no `<style>` blocks or external links (GitHub SVG safe)
- Font stack: `'Courier New', 'Lucida Console', monospace`
- `assets/banner.svg` committed to repo, ready for README `<img>` tag
